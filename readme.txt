Gebruikte ontwikkelomgeving:
MAMP (Mac, Apache, MySQL, and PHP) server die gebruikt is, is AMPPS (version 4.3). AMPPS is een stack van Apache, MySQL, Perl and Python. Het maakt gebruik van PHP 7.4 en MySQL 8.0.31.
Deze is hier te downloaden: https://ampps.com/downloads/

Gebruikte Code Editor: 
Visual Studio Code (VSC) van Microsoft
Deze is hier te downloaden: https://code.visualstudio.com/

Applicatie

Voor het kunnen openen van de applicatie moet de map ‘Ledenadministratie’ op een eigen development server geplaatst worden. Ga vervolgens naar de map ‘model’ en open het bestand ‘login.php’. Voer hier de gegevens in om verbinding te kunnen maken met een eigen database. Importeer ‘ledenadministratie.sql’ in deze database. Deze is te vinden in de map 'Ledenadministratie'. Vervolgens kan index.php geopend worden in een browser.

De applicatie start met een inlogscherm. In de database zijn twee gebruikers aangemaakt, waarmee kan worden ingelogd:
1. Gebruikersnaam: secretaris
   Wachtwoord: wwsecr
2. Gebruikersnaam: penningmeester
   Wachtwoord: wwpenm

Deze gebruikers zijn aangemaakt met het bestand create_users.php. Hiermee kunnen eventueel ook nieuwe gebruikers worden aangemaakt.

Zie voor meer informatie het verslag 'Eindopdracht PHP & MySQL.docx' en de video 'Demonstratie Applicatie Ledenadministratie.mp4'. Deze zijn te vinden in de map 'Eindopdracht'.
